# Privacy Policy

**Last updated:** July 2026

## Rase Filter Privacy Policy

Rase Filter is designed to help users filter and research YouTube Shorts
more efficiently.

We respect your privacy.

## Information We Collect

Rase Filter does **not** collect, store, transmit, sell or share any
personal information.

The extension does not require user accounts and does not collect:

-   Name
-   Email address
-   Passwords
-   Payment information
-   Location
-   Browsing history outside of YouTube
-   Personally identifiable information

## Local Storage

The extension stores the following information locally in your browser
using the Chrome Storage API:

-   Filter preferences
-   Selected language
-   Extension settings
-   Local saved videos library

This information never leaves your device.

## Permissions

### Storage

Used to save user preferences and extension settings.

### Side Panel

Used to display the extension interface while browsing YouTube Shorts.

### YouTube Access

The extension accesses YouTube Shorts pages only to:

-   Read publicly available video information
-   Apply user-selected filters
-   Display extension features

No personal data is collected or transmitted.

## Third Parties

Rase Filter does not sell, share or transfer user data to third parties.

## Remote Code

Rase Filter does not execute remote code.

## Data Security

All stored data remains on the user's device.

No external servers are used.

## Changes

This Privacy Policy may be updated as the extension evolves.
