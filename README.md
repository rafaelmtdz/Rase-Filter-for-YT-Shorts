# Rase Filter 3.1 Stable+

Reconstruída usando a lógica estável da versão 3.1.

## Incluído
- Filtro por views, idade, idioma e anúncios da base estável.
- Limite de duração ajustável de 1 a 180 segundos, padrão em 28 segundos.
- Controle de som no painel lateral, com volume de 0% a 100% e botão de mudo.
- Alterações de som não reiniciam nem reanalisam o Short.
- Sliders de duração e volume usam gravação com pequeno debounce para evitar excesso de eventos.
- HUD posicionado fora do vídeo quando há espaço.
- HUD não captura cliques; somente a estrela de salvar é clicável.

## Instalação
1. Extraia a pasta.
2. Abra `chrome://extensions`.
3. Ative o modo do desenvolvedor.
4. Remova/desative a versão anterior.
5. Clique em **Carregar sem compactação** e selecione esta pasta.
6. Feche as abas antigas do YouTube e abra uma nova.
